# Être Patisserie - Order Management System

A Next.js-based order management system for Être Patisserie.

## Quick Start

```bash
pnpm install    # Install dependencies
pnpm dev        # Start development server on port 3001
pnpm build      # Production build
pnpm start      # Start production server on port 3001
```

## Tech Stack

- **Framework**: Next.js 16, React 19, TypeScript 5
- **UI**: Radix UI, shadcn/ui, Tailwind CSS v4
- **Forms**: React Hook Form, Zod
- **Package Manager**: pnpm

## Deployment

This project is configured to deploy on Vercel. See the deployment section below for instructions.

## How to Deploy

1. Push this repository to GitHub
2. Connect your GitHub repository to Vercel
3. Deploy automatically with each push to main
